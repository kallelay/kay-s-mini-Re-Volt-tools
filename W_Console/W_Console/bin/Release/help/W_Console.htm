Texture animation

 

Texture animation file

Texture animation file is an ASCII-text file which structure is like the following:

Frame 0
Texture: G
Time: 0,01
uv1: 0,12x0,54
uv2: 0x0,57
uv3: 0x0,5
uv4: 0,2x0,5
End Frame

Frame 1
Texture: G
Time: 0,01
uv1: 0,11934x0,54
uv2: -0,00066x0,57
uv3: -0,00066x0,5
uv4: 0,19934x0,5
End Frame

 

More general:

Frame N
Texture: <Tex>
Time: 0.001
uv1: UxV
uv2: UxV
uv3: UxV
uv4: UxV
End frame

Where U and V are two float numbers from 0~1
